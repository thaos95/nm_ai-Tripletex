# nm-ai2

`nm-ai2` er et enkelt Python-prosjekt satt opp som en ren baseline for videre utvikling.

## Krav

- Python 3.11 eller nyere

## Installere avhengigheter

Prosjektet har ingen eksterne avhengigheter enda, men du kan fortsatt bruke:

```bash
pip install -r requirements.txt
```

## Kjøre prosjektet

Kjør prosjektet fra prosjektmappen:

```bash
python main.py
```

Du skal da se en enkel oppstartsmelding i terminalen.
